# Custom Shop API

1. clone repository
2. Installation

```bash
$ npm install
```

3. Clone archive ```.env.template```, rename to: ```.env``` & complete necesary informacion

4. config Docker Database.

```bash
# Up Docker image
$ docker-compose up -d
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
